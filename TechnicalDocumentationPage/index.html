Introduction

Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly, procedural), object-oriented, and functional programming.

print("Hello, World!")

Key features of Python include:

  • Easy to learn with simple syntax
  • Cross-platform compatibility
  • Extensive standard library
Python Basics

Python has a simple syntax similar to the English language. Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.

Variables are created when you assign a value to them:

x = 5 y = "Hello"

Python has the following data types built-in by default:

  • Text Type: str
  • Numeric Types: int, float, complex
  • Boolean Type: bool
Control Flow

Python supports the usual logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Greater than: a > b

Python relies on indentation (whitespace at the beginning of a line) to define scope in the code. Other programming languages often use curly-brackets for this purpose.

if x > 10: print("x is greater than 10") else: print("x is 10 or less")

Loops in Python:

for x in range(5): print(x)
Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. A function can return data as a result.

def my_function(): print("Hello from a function")

To call a function, use the function name followed by parenthesis:

my_function()

Functions can have parameters:

def greet(name): print("Hello, " + name) greet("Alice")

Functions can return values using the return statement:

def square(x): return x * x
Data Structures

Python has several built-in data structures:

  • Lists: Ordered, mutable collections
  • Tuples: Ordered, immutable collections
  • Dictionaries: Key-value pairs
  • Sets: Unordered collections of unique elements

Example of a list:

fruits = ["apple", "banana", "cherry"] print(fruits[1]) # Outputs: banana

Example of a dictionary:

person = { "name": "John", "age": 30 } print(person["name"]) # Outputs: John
